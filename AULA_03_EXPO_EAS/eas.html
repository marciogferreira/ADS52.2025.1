O que é Expo EAS?

Expo Application Services (EAS) é a solução da Expo para build, atualização e deploy de apps.

Criando uma Conta no Expo.dev

  • Expo.dev - Acesse o site expo.dev e crie sua conta.
  • Subindo seu App - Instale o EAS no seu PC: npm install -g eas-cli.
  • EAS Login - Execute o comando no seu terminal: npx eas login
  • Credenciais - Informe as mesmas credenciais do seu Expo.dev
  • Subindo App - Execute no CMD o comando npx eas init
  • Subindo Atualizações do App - Execute no CMD o comando npx eas update --branch preview --message "Comentário"

Principais Serviços do EAS

  • EAS Build - Compila apps para Android e iOS na nuvem.
  • EAS Submit - Facilita o envio de apps para as lojas.
  • EAS Update - Permite atualizações OTA (Over-The-Air).
  • EAS Metadata - Gerencia metadados das lojas.

Vantagens do EAS

  • Elimina necessidade de Xcode ou Android Studio.
  • Permite builds para múltiplas plataformas simultaneamente.
  • Reduz o tempo de entrega de novas versões.
  • Facilita atualizações sem precisar de uma nova build.

Como Começar?

npx expo install -g eas-cli
    eas build:configure
    eas build -p android

Comandos básicos para configurar e iniciar um build.

Conclusão

Expo EAS simplifica o ciclo de vida de um app React Native, tornando o desenvolvimento e deploy mais rápidos e eficientes.
